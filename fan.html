"What must the fan OVERCOME?"

This is the SUM of all pressure drops in the system. You must calculate it.

Dos & Don'ts (AMCA 201)
Installation is KEY
DO: Use VFDs for control (not dampers).
DO: Install 5-10 diameters of straight duct at the INLET.
DO: Use vibration isolators.
DON'T: Block the inlet (causes swirl).
DON'T: Run in "Stall" (unstable flow).
