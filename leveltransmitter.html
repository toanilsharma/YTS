Where does this live?
NOT in the Transmitter!
99% of the time:
The calculation is done in the DCS or PLC. (e.g., a "CHAR" Function Block)
Transmitter: Measures 0-100% *Level* (Pressure) and sends a 4-20mA signal.
DCS/PLC: Receives 4-20mA, applies the strapping table, and displays % *Volume*.

Stilling Wells
Got Turbulence?
For tanks with agitators, mixers, or turbulent inlets, you MUST install the sensor (GWR/Radar/DP) in a stilling well.

Expert Detail: DP
Vapor Space Comp
If using a DP sensor on a closed tank, the "Low" side of the sensor MUST be connected to the tank's vapor space.
This subtracts the blanket gas pressure, otherwise a 1 PSI pressure change looks like a 27.7" level change!
